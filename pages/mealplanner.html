WhatDoIEat

Ein KI-Essensplaner, der auf dem NUC-Server als Hermes-Agent läuft. Er plant autonom vegetarische Wochenpläne und schreibt Mahlzeiten samt Einkaufsliste direkt in Notion – per Telegram-Befehl.

Konzept

Die Idee

Jede Woche die gleiche Frage: „Was essen wir?" – WhatDoIEat löst das Problem automatisiert. Ein KI-Agent wählt aus über 120 kuratierten vegetarischen Gerichten, vermeidet Wiederholungen und generiert eine komplette Einkaufsliste für 2 Personen.

Infrastruktur

NUC Server

  • Hermes AI Agent
  • Telegram Bot Interface
  • OpenRouter API Gateway
  • Claude 4.5 Haiku (LLM)
  • Notion MCP Server (npx)
Features

Was der Agent kann

  • Wochenplan Mo–So generieren
  • Einkaufsliste mit Mengen
  • Notion-Seite automatisch erstellen
  • Wiederholungsvermeidung (History)
  • Neue Gerichte vorschlagen
Pipeline

Architektur — Vom Telegram-Befehl zur Notion-Seite

User: "Plane die Woche" → Telegram Bot │ ▼ [1] Hermes Agent (NUC Server) │ Skill: meal_planner ▼ [2] Read gerichte.md + history.json │ 120+ Gerichte laden, letzte 3 Wochen ausschließen ▼ [3] LLM Call → OpenRouter → Claude 4.5 Haiku │ 7 Mahlzeiten + Einkaufsliste generieren ▼ [4] Notion MCP → Seite erstellen │ Tabelle: Wochenplan | To-Dos: Einkaufsliste ▼ [5] Update history.json + Antwort via Telegram
Stack
